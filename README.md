# 小马考试通ponyexam（VUE版本）

#### 介绍
小马考试通（VUE版本）主要是实现考试数字化而进行开发的信息化系统，采用Springboot+Vue技术研发，支持包括单选题、多选题、判断题、填空题、问答题、综合题等各类常用题型。采用系统智能组卷方式、支持人工和自动两种阅卷方式，从而实现考试自动化和数字化，大大的提高了考试工作效率，极大降低人力工作量。适合在各类学校、培训机构、单位内训等场景使用或二次开发。
